OrderedDictionary

A dictionary that remembers the order in which keys were first inserted.

Creates a ordered dictionary wrapped within metamethods.

  • __index -> :get(key)
  • __newindex -> :set(key, value)
  • __len -> :getLength()
  • __iter -> :iterate()
  • __call -> unwraps: OrderedDictionary<K, V>
